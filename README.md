# MeatyEFTRelease

Advanced external radar and companion toolkit for **Escape from Tarkov** focused on performance, clean visuals, and live data features.

You can find out more on https://discord.gg/GXmBagdP5s and get support, pass on suggestions etc etc

Please note, dlcks will be banned or ignored, use your brain

---

# Features

- Real-time radar rendering  
- Player tracking  
- Loot filtering & wishlist system  
- Quest helper / objective markers  
- Extract locations  
- Group detection  
- Map zoom / pan / follow modes  
- Live player data integration  
- Dogtag database integration  
- Fast scatter-read architecture  
- Optimised C++ performance build  

---

# Tech Stack

- C++  
- DirectX9  
- ImGui  
- GLM  
- nlohmann::json  
- libcurl
